God Mode

God Mode is a collection of customization features that allow you to kind of do whatever you want. If you enable God Mode, you get access to the following features (which show up in the game as purple text):

  • Create custom players by going to Tools > Create A Player
  • Edit any player by going to their player page and clicking Edit Player
  • Force any trade to be accepted by checking the Force Trade checkbox before proposing a trade
  • Disable regional three player minimum.
  • You can become the GM of another team at any time
  • You will never be fired!
  • Some custom roster have much higher rated players. In order to improve performance with those you can also make player rating matter much more by going to Tools > Custom Roster Mode
  • You can also use this mode for normal randomly generated roster to make the game play a little more like actual Pro LOL. Currently LOL GM is a little more random in order to make the game more difficult.

However, if you enable God Mode within a league, you will not get credit for any Achievements. This persists even if you disable God Mode. The only way to get Achievements is in a league where God Mode has never been enabled.
